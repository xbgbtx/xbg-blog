Artist Map

2021-04-28
#web  #music  #programming  #map  #wikipedia 

This interactive map shows the birth and formation location of nearly sixty thousand musicians and musical ensembles.

Try it in your browser!

View the Source Code

Leaflet JS Map⌗

The map data is displayed using Leaflet JS - a flexible front end for map data. The map tiles are sourced from Open Street Map.

WikiData⌗

Data is queried from WikiData a service that enables SPARQL queries on a database generated from the article text of Wikipedia articles.
