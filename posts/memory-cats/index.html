Memory Cats

2022-02-27
#web  #cards  #programming  #lit  #javascript 

Memory cats is a memory card game that sources its images from an API so that every game the player sees new cats.

Try it in your browser!

View the Source Code

Frontend Framework⌗

The frontend of the system uses the following technologies:

  • Lit - Front end framework using the open web components standard.
  • xstate - Provides declarative state machines for controlling application behaviour.

Application Back End⌗

The application gets the cat pictures from The Cat API.

In order to use the API without revealing the private API key a wrapper API written in Go is used.
